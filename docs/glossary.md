# Glossary — S.E.A.M Protocol

## English

| Term | Definition |
|------|-----------|
| **AWG** | Atmospheric Water Generator — extracts drinking water from humidity |
| **BMS** | Battery Management System — monitors and protects lithium cells |
| **Bio-Leaching** | Using bacteria and fungi to extract metals from spent batteries |
| **DePIN** | Decentralized Physical Infrastructure Network |
| **DS** | Directional Solidification — purification technique for silicon |
| **E2E** | End-to-End encryption |
| **ESG** | Environmental, Social, and Governance compliance certificate |
| **FaaS** | Farming-as-a-Service — protocol-managed autonomous agriculture |
| **LFP** | Lithium Iron Phosphate — battery chemistry used across all systems |
| **LoRa** | Long Range — low-power radio protocol for IoT communications |
| **LoRaWAN** | LoRa Wide Area Network — network protocol on top of LoRa |
| **Mesh-ID** | Decentralized citizen identity on the LoRa mesh network |
| **MPPT** | Maximum Power Point Tracking — optimizes solar panel output |
| **NDVI** | Normalized Difference Vegetation Index — crop health metric |
| **PAYGo** | Pay-As-You-Go — zero upfront cost, pay via usage |
| **PV** | Photovoltaic — solar panels |
| **SFRES** | S.E.A.M Factory & Renewable Energy Station |
| **UBI** | Universal Basic Income — distributed from agricultural surplus |
| **UMG-Si** | Upgraded Metallurgical-Grade Silicon |
| **V2G** | Vehicle-to-Grid — EVs discharge stored energy back to the grid |

## العربية (Arabic)

| المصطلح | التعريف |
|---------|---------|
| **مولد مياه جوي (AWG)** | يستخرج مياه الشرب من الرطوبة الجوية |
| **نظام إدارة البطاريات (BMS)** | يراقب ويحمي خلايا الليثيوم |
| **الترشيح الحيوي** | استخدام بكتيريا وفطريات لاستخراج المعادن من البطاريات المستنفدة |
| **الزراعة كخدمة (FaaS)** | زراعة مستقلة يديرها البروتوكول |
| **ليثيوم حديد فوسفات (LFP)** | كيمياء البطارية المستخدمة في جميع الأنظمة |
| **شبكة لورا** | بروتوكول راديو منخفض الطاقة للاتصالات |
| **هوية الشبكة (Mesh-ID)** | هوية المواطن اللامركزية على شبكة لورا |
| **الدخل الأساسي الشامل (UBI)** | يوزع من فائض الإنتاج الزراعي |
| **سيليكون معدني محسّن (UMG-Si)** | سيليكون منقى من الرمال |
| **المركبة إلى الشبكة (V2G)** | المركبات الكهربائية تفرغ الطاقة المخزنة للشبكة |
