# PulseBus — Mass Transit EV

> Heavy-duty electric bus for inter-city and intra-city transit.

## Specifications

| Parameter | Value |
|-----------|-------|
| Battery | 4 × 50 kWh blocks (200 kWh total) |
| Range | 250 km per swap |
| Connectivity | Mobile LoRa mesh repeater |
| V2G | Full bidirectional support |
| Track | B (deploys after >5,000 vehicle fleet threshold) |

## Design Notes
- Uses the same Skateboard chassis as PulseVan
- Battery blocks are standardized and swappable at SFRES stations
- Acts as a mobile energy storage unit when parked (V2G)

## Status
🔴 **Not started** — Track B R&D. Requires fleet density before deployment.
