# Battery Swap Module

> Standardized hot-swap mechanism for all S.E.A.M vehicles and stations.

## Requirements

- Sub-3-minute full swap
- 200A continuous connector with arc suppression
- Mechanical interlock (vehicle must be stationary)
- Compatible with PulseTrike (15 kWh), PulseVan (modular), PulseBus (4×50 kWh)
- E-STOP accessible from all sides
