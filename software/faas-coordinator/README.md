# FaaS Coordinator — Farming as a Service

> Orchestrates the autonomous agricultural swarm on behalf of landowners.

## Workflow
1. Landowner draws boundaries on Mesh-ID
2. Ownership registered on-chain
3. Press "Deploy Swarm"
4. Protocol handles planting → harvesting → delivery

## Yield Distribution
- 80% → Landowner
- 15% → Protocol logistics (maintenance, recycling)
- 5% → UBI Fund (Grain Tokens)
