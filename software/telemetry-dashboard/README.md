# Telemetry Dashboard

> Real-time monitoring for energy grid, fleet, and agricultural operations.

## Features
- [ ] Live energy production/consumption per SolarSync node
- [ ] Fleet tracking (PulseTrike, PulseVan, PulseBus)
- [ ] Battery SOC heat maps
- [ ] Agricultural swarm status (Phase 3)
- [ ] Alert system for grid emergencies

## Tech Stack (Proposed)
- Frontend: React/Next.js
- Backend: Python (FastAPI)
- Database: TimescaleDB (time-series)
- Maps: Leaflet/Mapbox
