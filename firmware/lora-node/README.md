# LoRa Node Firmware

> Firmware for individual LoRa sensor/telemetry nodes deployed at each SolarSync site.

## Target
- **MCU**: STM32WL (integrated LoRa) or ESP32 + SX1262
- **Frequency**: 868/915 MHz
- **Protocol**: LoRaWAN Class A (battery-optimized)

## Features
- [ ] Sensor data collection (voltage, current, temperature, soil moisture)
- [ ] LoRaWAN join and uplink
- [ ] AES-128 E2E encryption
- [ ] Deep sleep power management (< 5 µA idle)
- [ ] Adaptive data rate (ADR)
