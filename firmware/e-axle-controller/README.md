# e-Axle Motor Controller Firmware

> FOC (Field-Oriented Control) firmware for the axial flux motors.

## Features
- [ ] Sensorless FOC algorithm
- [ ] Regenerative braking
- [ ] Thermal derating (desert heat management)
- [ ] CAN bus interface to vehicle controller
- [ ] Torque vectoring support (dual-motor configurations)
