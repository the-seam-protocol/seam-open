# Battery Management System Firmware

> Universal BMS firmware for LFP packs across all S.E.A.M systems.

## Features
- [ ] Per-cell voltage monitoring (16S for SolarSync, configurable)
- [ ] Passive cell balancing
- [ ] Temperature monitoring with thermal cutoff
- [ ] SOC estimation (Coulomb counting + OCV)
- [ ] CAN bus communication to host system
- [ ] Safety interlocks (over-voltage, under-voltage, over-current, over-temp)
