# Battery Swap Controller Firmware

> Controls the mechanical swap station and validates battery modules.

## Features
- [ ] Mechanical latch engage/disengage sequence
- [ ] Battery module authentication (cryptographic handshake)
- [ ] SOC readback and health check before release
- [ ] Interlock: vehicle must be stationary and in park
- [ ] E-STOP monitoring
