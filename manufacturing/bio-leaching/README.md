# Bio-Leaching — Battery Recycling

> Biological battery recycling with 90%+ material recovery.

## Biological Agents
- **Acidithiobacillus ferrooxidans** — Iron-oxidizing bacteria
- **Aspergillus niger** — Citric acid-producing fungi

## Organic Acid Sources
- Sugarcane waste → fermentation → citric acid
- Citrus waste → fermentation → gluconic acid
- **Zero imported chemicals**

## Reaction
```
LiFePO₄ + organic acid + bacteria → Li⁺ + Fe³⁺ + PO₄³⁻ → direct reuse in assembly lines
```

## Recovery Rate
**90%+** — Closed resource loop
