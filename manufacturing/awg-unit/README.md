# Atmospheric Water Generator (AWG)

> Extracts 20–30 liters of drinking water per day from atmospheric humidity.

## Specifications

| Parameter | Value |
|-----------|-------|
| Output | 20–30 L/day |
| Power Source | SolarSync DC bus |
| Climate Suitability | Arid/semi-arid (Sudan) |
| Filtration | Multi-stage (sediment + carbon + UV) |

## Status
🔴 **Not started** — Requires humidity feasibility study for target Sudanese regions.
